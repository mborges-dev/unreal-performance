# Changelog

All notable changes to the UNREAL Performance marketing site documented here.

## [0.1.0] — 2026-06-04

Initial public source release. Marketing site for the UNREAL Performance studio.

### Added

- Single-page marketing site (Next.js 16 App Router)
- Custom typography: Fraunces (display) · DM Sans (body) · IBM Plex Mono (technical)
- Smooth-scroll via Lenis
- Page transitions and reveal animations via Framer Motion 12
- Custom `@theme inline` Tailwind v4 configuration

### Stack

Next.js 16 · React 19 · TypeScript (strict) · Tailwind CSS v4 · Framer Motion 12 · Lenis

### Deployment

Built and deployed via Vercel.
